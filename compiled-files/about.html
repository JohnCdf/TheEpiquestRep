About Epiquest

Languages: HTML/PUG, LESS/CSS, JS, COFFEESCRIPT
Technologies: Github, Gulp, Keybase

Epiquest is a simple web adventure game made in the span of 3 months, originally created by JohnCdf, it was later picked up and worked on by DarkHole - together, with the use of frameworks and precompiled languages, the game was given life. The repository for the game can be found here.
